# Document Link

> Listen for link clicks

## Install

```
$ npm install @document/link
```

## Usage

```js
const documentLink = require('@document/link');

documentLink(() => {

});
```

## API

### documentLink(callback, [root])

#### callback

Type: `function`

Will be executed on link click.

#### root

Type: `object`

The root node.
